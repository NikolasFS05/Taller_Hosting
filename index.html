¿Que es JavaScript?

Hoy en día las páginas web están construidas bajo las etiquetas HTML y los estilos CSS, sin embargo, el uso de solo estos dos hace que el sitio se limite en ciertas acciones dinámicas que no se pueden controlar con solo el código de etiquetas y estilos, en esta parte es donde entra el lenguaje de JavaScript.
“JavaScript es un lenguaje de programación o de secuencias de comandos que permiten implementar funciones complejas en páginas web” según se dice en [1], [2]. Este lenguaje en si es utilizado para hacer que la información y botones que están contenidos en la página dejen de estar estáticos y se transformen en acciones graficas las cuales generen una interactividad mayor para las personas que hagan uso del sitio.
